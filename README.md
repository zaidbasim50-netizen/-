# Aurora Beauty Website

موقع إلكتروني متخصص بمنتجات العناية بالبشرة، يهدف إلى تقديم تجربة تسوق سهلة وجذابة لمنتجات **Aurora Beauty**.

## 📂 هيكل المشروع
- `index.html` : الصفحة الرئيسية.
- `product.html` : صفحة المنتجات والعروض.
- `care.html` : صفحة العناية والدعم.
- `style.css` : تنسيقات وتصميم الموقع.
- `script.js` : وظائف تفاعلية للموقع.
- `data.js` : بيانات المنتجات أو المحتوى.
- `.gitignore` : إعدادات تجاهل الملفات.
